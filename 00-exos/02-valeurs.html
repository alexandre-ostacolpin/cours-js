Observez le code ci-dessous et tentez de prévoir les valeurs finales de chaque variable à la fin de chaque ligne(par exemple après la ligne 'var a = 2;'si je fait un console.log de a j'aurai 2 comme résultat. cf.capture d'écran

L'idée est de faire les calculs vous-mêmes, sur une feuille par axemple AVANT de regarder dans le console !

var a = 2; Exemple = 2
a = a -1; Exenple (2-1) = 1
a ++; Exenple (1+1) = 2
var b = 8; Exenple = 8
b += 2; Exenple (8+2)= 10
var c = a + b * b; Exemple (10x10+2) = 102
var d = a * b + b; Exemple (2x10=20+10) = 30
var e= a * (b + b); Exemple (10+10=20x2) = 40
var f = a * b / a; Exemple (10/2=5x2) = 10
var g = b / a * a; Exemple (10/2=2 * 2) = 10
