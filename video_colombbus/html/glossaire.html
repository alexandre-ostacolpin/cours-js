glossaire

Binaire
fichier lisible et exécutable par l'ordinateur
caster
opération de conversion d'un type de données dans un autre
chaîne vide
Un caractère signifiant qu'une chaîne est vide
console
zone permettant de consulter les messages produits par JS
concaténer
rabouter 2 chaînes de caractère en une seule
Compilé
Analysé un code et fabriquer un binaire
déclaration
action de créer quelque chose en programmation : concernera souvent les variables
échapper des caractères
enlever sa fonction à un caractère afin de simplement l'afficher
instruction
unité de code à lancer; fonctions souvant ligne par ligne
Interpréter
Analysé un code et le faire exécuter à l'odinateur
saut de ligne
un caractère signifiant que le logiciel doit afficher un retour à la ligne
variable
élément de programmation qui va servir à contenir quelque chose : nombre, chaîne, etc. et sera enregistré au moyen d'un nom.
ex : let bidule = 12
