JS - III 05 IF Les opérateurs logiques


Ce type d'opérateurs va nous permettre d'effectuer plusieur comparaisons dans nos tests ou d'inverser la valeur logique du résultat d'un test.

1- Présentation et utilisation des opérateurs logiques

Le JS supporte 3 opérateur logiques : l'opérateur logique "ET", "OU" et "NON".

OPERATEURS LOGIQUES
Opérateurs, nom Opérateur, symbole Description
AND(ET) && renvoie true si toute les comparaisons sont évaluées à true ou false sinon
OR(OU) || Renvoie true, si au moins l'une des comparaisons est évaluée à true, ou false sinon
NO(non) ! Renvoie false si une comparaison est évaluée à true, ou renvoie true dans le cas contraire
