JS - III 03 IF else


1- if...else...en JS

La structure conditionelle if...else (en français si... sinon) sera plus complète car on va exécuter un bloc de code si un test renvoie true ou un autre dans le cas contraire.

if..else...

  • if(x > 1){
    alert('x contient une valeur supérieur à 1');
    }else{
    alert('x contient une valeur inférieur à 1');
    }
